# Galois Field

## definition

Let
$p \in P$
,
characterize galois field (finite field)
$G$
by
$\Z / p \Z$
.

### pow

It takes
$O(\log \lbrace P + n \rbrace)$
time complexity.

### inv

It takes
$O(\log P)$
time complexity.

### +, -, *

It takes 
$O(1)$
time complexity.

### /

It takes
$O(\log P)$
time complexity.
